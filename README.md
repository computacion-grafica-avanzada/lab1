# Photon Mapper using OptiX 7.2 and CUDA 11.1

Based on [Siggraph 2019/2020 OptiX 7/7.1 Course Tutorial Code](https://github.com/ingowald/optix7course)

## Configuration guide params.txt

- .obj file path
- camera position
- camera up
- light position
- light direction
- light intensity
- max number of photons
- max recursion depth
- max radius search
- antialiasing level
